# Unit-Converter
A simple and user-friendly web app that converts between metric and imperial units. It helps users quickly convert measurements for length, volume, and mass

# Features:

Convert between meters ↔ feet, liters ↔ gallons, and kilograms ↔ pounds.

Instant conversion results with a clean and minimal UI.

Responsive design that works on desktop and mobile.

Built using HTML, CSS, and JavaScript.

# Example Conversions:

22 meters = 72.178 feet.

22 liters = 83.279 gallons.

22 kilos = 48.502 pounds.
